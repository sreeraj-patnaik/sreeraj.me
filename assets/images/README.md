# Portfolio images

Add your photos here. The site uses these filenames (fallbacks show initials or icons if missing):

| File | Where it appears |
|------|------------------|
| `avatar.jpg` | Hero section — profile/headshot (circular) |
| `about.jpg` | About section — portrait or casual photo (4:5 ratio works well) |
| `education.jpg` | Education section — campus, graduation, or study photo (3:4 ratio) |
| `sql-agent.jpg` | Projects — SQL Agent AI project screenshot or banner (optional) |

Recommended: JPG or WebP, at least 400px on the short side. The layout is responsive and uses `object-fit: cover`.
